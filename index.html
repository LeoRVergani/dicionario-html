Dicionário de Tags HTML

  • <!DOCTYPE>

    Define o tipo de documento e a versão do HTML que está sendo usada. Deve ser a primeira linha em um documento HTML.

    <!DOCTYPE html>

  • <html>

    Elemento raiz de um documento HTML. Envolve todo o conteúdo da página.

    <html></html>

  • <head>

    Contém metadados/informações sobre o documento. Inclui elementos como <title>, <meta>, <link>, <style>, etc.

    <head></head>

  • <title>

    Define o título do documento, que aparece na aba do navegador.

    <title>Título da Página</title>

  • <meta>

    Define metadados sobre o documento HTML, como charset, autor, descrições, etc.

    <meta charset="UTF-8">

  • <body>

    Contém todo o conteúdo visível da página, como textos, imagens, vídeos, etc.

    <body></body>

  • <h1> a <h6>

    Definem cabeçalhos/títulos, sendo <h1> o mais importante e <h6> o menos importante.

    <h1>Cabeçalho 1</h1>

  • <p>

    Define um parágrafo.

    <p>Este é um parágrafo.</p>

  • <a>

    Define um hiperlink para outra página ou recurso.

    <a href="https://example.com">Visite o Example</a>

  • <img>

    Incorpora uma imagem no documento.

    <img src="imagem.jpg" alt="Descrição da imagem">

  • <ul>, <ol> e <li>

    Define listas não ordenadas (<ul>), listas ordenadas (<ol>) e itens de lista (<li>).

    <ul>
      <li>Item da lista</li>
    </ul>

  • <div>

    Define uma seção ou um contêiner genérico em um documento.

    <div>Conteúdo</div>

  • <span>

    Define uma seção genérica em um documento, usada para estilização.

    <span>Texto em linha</span>

  • <form>

    Define um formulário para entrada de dados do usuário.

    <form action="/submit" method="post"></form>

  • <input>

    Define um campo de entrada de dados em um formulário.

    <input type="text" name="nome">

  • <button>

    Define um botão clicável.

    <button>Clique aqui</button>

  • <table>, <tr>, <th> e <td>

    Define uma tabela (<table>), uma linha de tabela (<tr>), um cabeçalho de célula (<th>) e uma célula de tabela (<td>).

    <table>
      <tr>
        <th>Cabeçalho</th>
        <td>Dado</td>
      </tr>
    </table>

  • <style>

    Define estilos CSS internos.

    <style>
      body {
        background-color: #f0f0f0;
      }
    </style>

  • <link>

    Define a ligação a um recurso externo, geralmente usado para vincular estilos CSS.

    <link rel="stylesheet" href="styles.css">

  • <script>

    Define um script, geralmente usado para adicionar JavaScript.

    <script src="script.js"></script>
